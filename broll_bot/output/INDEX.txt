HOW TO USE THESE FILES
================================================================================

FOR VIDEO EDITORS:
1. Open assignment_map.json
2. For each line_number (1-82):
   a. Find the corresponding line in script
   b. For each asset:
      - If type="broll": Cut clip from YouTube using start_time to end_time
      - If type="image": Place image in timeline
3. Layer visuals below narration (unless with_audio=true)
4. Use match_reasoning for editorial context

FOR DEVELOPERS/AUTOMATORS:
1. Parse assignment_map.json with JSON library
2. Extract video URLs and timestamps
3. Auto-download clips and trim to specified range
4. Place images from pic_number index
5. Check era_appropriate and with_audio flags

FOR PROJECT MANAGERS:
1. Review ANNOTATION_REPORT.txt for statistics
2. Check coverage metrics (100% complete)
3. Reference top video sources
4. Review recommendations for expansion

FOR RESEARCHERS:
1. Analyze match_reasoning fields for annotation patterns
2. Review relevance_score distribution

4. Reference ASSIGNMENT_MAP_README.txt for methodology
